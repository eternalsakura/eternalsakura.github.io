Mac上本机和虚拟机终端代理设置

发表于 2018-02-02 | 分类于 杂项 | |

Mac终端代理

vim ~/.bash_profile
然后在里面输入

export ALL_PROXY=socks5://127.0.0.1:1086,这里的1086根据自己的电脑去修改，设置如下

可以直接把上述代码拷贝到上述对应文件的末尾，修改完成之后执行如下命令使修改生效
source ~/.bash_profile
然后只要输入setproxy就可以设置代理，输入unsetproxy就取消代理
设置完毕后，输入curl -i http://ip.cn确认修改成功。
